1. Manage Transaction Parts and their Functionality


A. Transaction Types

These buttons, if clicked, shows all the certain transactions on the Transaction List below. These buttons will serve as a filter to show just specific transactions.

  1. All: Shows all the transactions that have been made.
  2. Processed: Shows all the transactions that have been processed.
  3. Pending: Shows all the transactions which are still pending.
  4. Cancelled: Shows all the transactions which have been cancelled.
  5. Complete: Shows all the transactions which have been approved/disapproved/completed.
B. Search Box

  • The search box can be used to find specific transaction by entering the transaction's name.
C. Selected Transaction Details

The Selected Transaction Details shows the summary of the selected transaction. The information will show by clicking a transaction on the transaction list.

i. Buttons

  1. Print: Clicking this button will print the selected transaction.
  2. View Tracking: To keep track of your claims.
  3. Edit Transaction:: Clicking this button will redirect you to the New Transaction form. Please refer to New Transaction manual/instruction.
  4. Cancel Transaction: Clicking this button will cancel the transaction.

    Note: You cannot cancel a transaction which have been completed.

ii. Date

  1. Date: To view claims tracking by date.

iii. Refresh Tracking Data

  1. Refresh Tracking Data: By clicking this button, the summary of the selected transaction will then refresh and shows new summary only if any changes were made.
D. No. of Records

  • By clicking this button, a list of numbers will drop down showing 10, 50, 100, 200, 500, and 1000 in which you need to select. This button will limit the number of transactions showed on the Transaction List. The default is a number of 100 records on the list.
E. Add New Transaction

  • By clicking the "Add New Transaction" button, you will be redirected to the New Transaction form. Please refer to New Transaction manual/instruction.
