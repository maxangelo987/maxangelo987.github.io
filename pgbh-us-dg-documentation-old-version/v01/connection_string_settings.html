Connection String Settings



Note: Please fill up all required(*) fields.

For figure 1.5:

  • Arrow L: Input here the specified server.
  • Arrow M: Input here the specified username.
  • Arrow N: Input here the specified password.
  • Arrow O: Input here the specified port.
  • Arrow P: Check the checkbox if you wish to change the database.
  • Arrow Q: Click this to test connection.
  • Arrow R: Click this after testing the connection to save the connection setting.
  • Arrow S: Click this to restart and apply the changes you've done.
  • Arrow T: Click this Cancel button when you wish not to change any of it.
  • Arrow U: This tab is for the Document Generator.
  • Arrow V: This tab is for the PAccO/Accounting.
  • Arrow W: This tab is for the PBMO/Budget.
  • Arrow X: This tab is for the Static Data.
