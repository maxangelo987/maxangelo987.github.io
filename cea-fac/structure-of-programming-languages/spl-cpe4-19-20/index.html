Course Description This course introduces programming languages. The description of a programming language is usually split into the two components of syntax (form) and semantics (meaning). In programming language theory, semantics is the field concerned with the rigorous mathematical study of the meaning of programming languages. In computer science, the syntax of a computer language is the set of rules that defines the combinations of symbols that are considered to be a correctly structured document or fragment in that language.
